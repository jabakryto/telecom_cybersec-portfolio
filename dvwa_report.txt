# DVWA Vulnerability Report

## 1. SQL Injection (Low)
- **Payload**: `' OR '1'='1`
- **Результат**: Получен список всех пользователей (admin, gordonb, 1337, pablo, smithy)
- **Вывод**: Приложение не использует параметризованные запросы → уязвимость подтверждена

## 2. Reflected XSS (Low)
- **Payload**: `<script>alert('XSS')</script>`
- **Результат**: Выполнено всплывающее окно в браузере
- **Вывод**: Входные данные не экранируются → возможна кража сессий

## 3. Default Credentials
- **Логин/пароль**: `admin` / `password`
- **Результат**: Успешный вход
- **Риск**: Использование дефолтных учётных данных

## 4. Результаты сканирования
- **Nikto**: см. `nikto_dvwa.txt`
  - Отсутствуют заголовки безопасности (X-Frame-Options, X-Content-Type-Options)
  - Обнаружена страница `/dvwa/login.php`
- **Dirb**: см. `dirb_dvwa.txt`
  - Найдены пути: `/dvwa/config/`, `/dvwa/hackable/`, `/dvwa/vulnerabilities/`
