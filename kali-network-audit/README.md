# README.md

## Проект: Сканирование и аудит локальной тестовой сети (Kali Linux)

**Краткое описание**
Проект демонстрирует процесс безопасного аудита изолированной тестовой сети с использованием Kali Linux и стандартных инструментов (nmap, netdiscover, nikto, enum4linux и т.д.). Цель — найти, задокументировать и классифицировать уязвимости, подготовить рекомендации по исправлению и оформить отчёт, пригодный для портфолио.

**Уровень:** Начальный

**Что в репозитории**

```
README.md                # Этот файл — краткое описание проекта и инструкции
report.md                # Шаблон отчёта, который нужно заполнить по результатам теста
lab/                     # (рекомендуется) файлы конфигурации VM / скрипты / wordlists
outputs/                 # сохранённые выводы nmap, nikto, enum4linux, pcap и т.д.
docs/                    # скриншоты, PDF-версия отчёта, презентация
```

---

## Требования

* Хост-машина с VirtualBox или VMware
* Kali Linux VM (рекомендуется последняя версия)
* Уязвимая VM: Metasploitable2/3, DVWA или аналог
* Сетевой режим: Host-Only или Internal Network (изолированная лаборатория)

---

## Быстрая установка окружения (VirtualBox)

1. Установи VirtualBox и расширения (Extension Pack).
2. Создай Kali VM (2 CPU, 4 GB RAM рекомендовано).
3. Разверни Metasploitable2 (OVF) или DVWA VM.
4. В настройках сети обеих VM выбери Host-Only Adapter (или Internal Network) и убедись, что они в одной подсети.
5. Сделай snapshot обеих VM до начала работ.

---

## Как запустить (коротко)

1. Запусти Kali и целевую VM в изолированной сети.
2. Определи IP-адреса командой `ip addr show` (на Kali) и `ifconfig`/`ip` на жертве.
3. Выполни сетевое обнаружение и сканирование (см. script.sh в lab/ или команды ниже).

Пример основных команд (копировать/вставить):

```bash
# Обновление
sudo apt update && sudo apt upgrade -y

# Определение интерфейсов
ip addr show

# Быстрое обнаружение хостов
sudo netdiscover -i <iface> -r 192.168.56.0/24

# Ping scan
sudo nmap -sn 192.168.56.0/24

# Полный скан сервисов и версий
sudo nmap -sS -sV -O -p- -T4 <target-ip> -oN outputs/nmap_full_<target-ip>.txt

# Nmap vuln scripts
sudo nmap -sV --script vuln <target-ip> -oN outputs/nmap_vuln_<target-ip>.txt

# Web scan
nikto -h http://<target-ip> -output outputs/nikto_<target-ip>.txt

# SMB enumeration
enum4linux -a <target-ip> > outputs/enum4linux_<target-ip>.txt

# Захват трафика
sudo tcpdump -i <iface> host <target-ip> -w outputs/capture_<target-ip>.pcap
```
